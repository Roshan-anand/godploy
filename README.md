# Godploy

Lightweight, single-binary, self-hosted alternative to Dokploy, Coolify. 

## Tech stack

* **Server:** Go
* **Runtime:** Docker
* **Ingress:** Traefik
* **DB:** SQLite (metadata)
* **UI:** Vue (SPA, served from binary)
