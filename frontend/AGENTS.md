# Agent Rules — Godploy Frontend

## Project Context

Vue 3 SPA for Godploy (self-hosted PaaS). Uses Bun for package management.

**Stack:** Vue 3 · TypeScript · Vite · Tailwind CSS v4 · Pinia · Vitest · Playwright

Use **bun** — not npm or yarn.

## Code Style Guidelines

Package Manager - Bun.
State Management - Pinia.
Component Lib - Shadcn/vue (located at ./src/components/ui/\*)
query - Tanstack Query (https://tanstack.com/query)
Form - Tanstack Forms (https://tanstack.com/forms)
Type Validation - Zod
